# Resume Dev

A modern resume builder built with Next.js and TailwindCSS.

## Features
- Responsive design
- Modern UI
- Easy customization
- Export to PDF

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Tech Stack
- Next.js
- TypeScript
- TailwindCSS
- React
